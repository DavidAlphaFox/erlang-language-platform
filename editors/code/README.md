# Erlang Language Platform

Provide support for the [Erlang](https://www.erlang.org/) Programming Language.

## Features

* Syntax Highlighting
* Go To Definition
* Find References
* Auto-completion
* Call Hierarchy
* Signature Help
* Diagnostics
* Inlay Hints
* ...
